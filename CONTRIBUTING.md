# Contributing

Thanks for contributing to RedTeamingAI.

## Getting started

1. Fork and clone this repository.
2. Create a feature branch.
3. Install dependencies with `npm install`.
4. Run `npm run build && npm test` before submitting.

## Pull requests

- Keep PRs scoped and focused.
- Add tests for behavior changes.
- Document API or UX changes in `README.md` where relevant.

## Coding guidelines

- TypeScript strict mode.
- Prefer explicit types over `any`.
- Keep handlers resilient (fail-open for proxy reliability where required).
